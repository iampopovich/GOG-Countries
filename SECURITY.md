# Security Policy

## Supported Versions

Security fixes are applied to the latest version on the default branch.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities.

Instead:

1. Open a private security advisory in this GitHub repository, or
2. Contact the maintainer through private channels.

Include:

- Affected component
- Reproduction steps
- Expected impact
- Suggested mitigation (if any)

Reports will be acknowledged promptly and addressed based on severity.
