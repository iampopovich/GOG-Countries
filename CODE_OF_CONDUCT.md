# Code of Conduct

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Our Standards

Positive behavior includes:

- Respectful and inclusive language
- Constructive feedback
- Empathy toward other contributors

Unacceptable behavior includes:

- Harassment, insults, or derogatory comments
- Personal attacks or intimidation
- Publishing others' private information without permission

## Enforcement

Maintainers will take appropriate corrective action in response to unacceptable behavior.  
Report issues via private security advisory or repository communication channels.
