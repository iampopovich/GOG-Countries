# Contributing

Thanks for contributing to `gog-countries`.

## Development Setup

```bash
git clone https://github.com/iampopovich/GOG-Countries.git
cd GOG-Countries
npm install
```

## Running Tests

```bash
npm test
npm run test:coverage
```

## Pull Request Guidelines

- Keep changes focused and small.
- Include a clear description: what changed and why.
- Update README if behavior or flags changed.
- All tests must pass (`npm test`).
- CI must be green before merge.

## Local Checklist

```bash
npm test
```

## Commit Guidelines

- Use descriptive commit messages.
- One logical change per commit.

## Reporting Issues

Use the issue templates in `.github/ISSUE_TEMPLATE/`.  
Include: reproduction steps, expected result, actual result, Node.js version.

## Scope

This project focuses on GOG price comparison across countries.  
Discuss major scope changes in an issue before implementing.
